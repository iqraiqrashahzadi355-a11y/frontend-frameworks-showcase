# Frontend Frameworks Showcase

A curated collection of starter projects built with today's most widely used frontend frameworks and libraries. Each project reflects the standard setup and structure produced by that framework's official tooling, offering a clean reference point for comparing developer experience across the modern frontend ecosystem.

## Frameworks Included

| Framework   | Folder                | Build Tool         |
|-------------|------------------------|---------------------|
| React       | `react-project`        | Vite                |
| Vue         | `vue-project`           | Vite                |
| Svelte      | `svelte-project`        | Vite                |
| Preact      | `preact-project`        | Vite                |
| Lit         | `lit-project`           | Vite                |
| Vanilla JS  | `vanilla-js-project`    | Vite                |
| Solid.js    | `solid-js-project`      | Vite (degit)        |
| Angular     | `angular-project`       | Angular CLI         |
| Ember       | `ember-project`         | Ember CLI           |
| Astro       | `astro-project`         | Astro CLI           |

## Getting Started

Each folder is a fully independent project. To run any of them locally:

```bash
cd <project-folder>
npm install
npm run dev
```

Angular and Ember use slightly different commands (`ng serve` and `npm start` respectively).

## About This Repository

This repository brings together starter implementations across multiple frontend frameworks, making it easy to explore and compare their conventions, folder structures, and tooling side by side.
